MCSA 70-410 Lab #0 - Intro

Today we will look at setting up a lab to study for the MCSA 70-410 exam. This lab is in no way meant to replace other methods of study, like books and training videos, but to supplement them. The first time I took the 70-410 exam, the PowerShell kicked my butt. Now, I know how to use PowerShell. I really enjoy automating day-to-day tasks in PowerShell. However, I did not know all of the commands associated with Active Directory, Windows Features, Group Policy, Firewall, Storage, DNS, DHCP, etc., as well as I thought.

It turns out, just reading books and watching videos didn’t quite do it for me. That is the purpose of this lab: to do everything we can think of that could possibly be covered in this exam in PowerShell.

Caveat: I have not passed the 70-410 yet. I’m writing this as I’m creating this lab and studying. Let’s begin.

Requirements:

  • Windows 8.1 or 10 Pro (This is so that you can run Hyper-V)
  • At least 30 GB of SSD space (You can use a hard drive, but it’s going to be slooowww)
  • A basic knowledge of Hyper-V and PowerShell
  • Server 2012 R2 iso (you should be able to sign up for a demo - no licenses needed)

I’m going to list a lot of complete PowerShell commands, but I strongly suggest you type them out instead of copying them. That’s the only way they’ll stick. In fact, after you start typing a command in PowerShell, like “Get-VM”, type a space, then a dash, and then start hitting the tab key. That will cycle through all of the parameters of the command and you’ll probably learn something very interesting.
